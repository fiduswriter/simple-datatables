Pass Bootstrap classes

According to this answer it is very difficult to pass bootstrap or any other classes to table elements in the new version of simple-datatables.

This example shows how to pass the .table-warning class to <tr>if members is 0.

Before you call simple-datatables you should add a .zeromember class to each span if members is 0.

After - using css selector - "#table_home tr:has(td > span.zeromember)" change styles.
